# AI 漫剧工作台 - 前端

基于 React + TypeScript + Ant Design 的 AI 漫剧创作平台前端。

## 功能模块

- AI 创作（小说生成、剧本生成）
- 剧本管理
- 资源准备
- 资源库
- 角色库
- 团队空间

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 技术栈

- React 18
- TypeScript
- Vite
- Ant Design 5
- Zustand (状态管理)
- React Router
- Axios
- Socket.IO Client
